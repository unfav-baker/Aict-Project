2.Variables

In C++, variables are used to store data values that can be manipulated during program execution. Each variable has a specific data type, such as int, float, or char, which defines the kind of data it can hold. Variables must be declared before use, specifying their type and optionally initializing them with a value.

4.Arrays

In C++, an array is a collection of elements of the same data type, stored in contiguous memory locations. Arrays are fixed in size, meaning the number of elements must be specified at the time of declaration. They allow efficient access to elements using an index, starting from zero.

5.Functions

In C++, a function is a block of code designed to perform a specific task, allowing code reuse and modularity. Functions are defined with a return type, a name, and optional parameters. They can return a value or perform actions without returning anything, depending on the return type.

6.Loops

In C++, loops are used to repeatedly execute a block of code based on a condition. The main types of loops are for, while, and do-while. A for loop is typically used when the number of iterations is known, while while and do-while loops are used when the number of iterations is determined by a condition.

7.If & If Else

In C++, the if statement is used to execute a block of code if a specified condition is true. The if-else statement allows you to execute one block of code if the condition is true, and a different block if the condition is false. Both are used for decision-making in programs.
