5. Functions in C++

Functions are blocks of code that perform specific tasks and can be reused throughout a program.

// Function declaration int add(int a, int b) { return a + b; } // Function with no return value void printMessage() { cout << "Hello World!"; } // Function overloading double add(double a, double b) { return a + b; } // Recursive function int factorial(int n) { if (n <= 1) return 1; return n * factorial(n - 1); }
